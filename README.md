# Layout Landing Page

A clean, modern, and fully responsive landing page built with modern web development tools and best practices. The project features a structured layout using advanced CSS/SCSS architecture, responsive design across multiple viewports, and clean code optimization.

## Technologies Used
* HTML5 / Semantic Layout
* SCSS (Sass) / BEM methodology
* JavaScript (ES6+ / Modules)
* Parcel (Bundler)
* Git & GitHub Pages

## Getting Started

Follow these instructions on how to set up the project locally:

1. Clone the repository:
   ```bash

   git clone [https://github.com/valerij0412/layout_landing_page.git](https://github.com/valerij0412/layout_landing_page.git)
   cd layout_landing_page

 Install dependencies:

Bash
npm install

Run the project locally:

Bash
npm start

Features

Fully responsive layout adapted for mobile, tablet, and desktop viewports.

Modular SCSS file structure with reusable mixins and variables.

Optimized assets and modern build pipeline via Parcel.
